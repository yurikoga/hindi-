# How to Create Login Form in HTML and CSS in Hindi

Um projeto simples de **formulário de login responsivo** usando HTML e CSS, com explicação em Hindi.

## ✨ Sobre

Este projeto demonstra como criar um **login form moderno e responsivo**, ideal para iniciantes, com conteúdo explicado em Hindi.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Layout responsivo
- Campos de login (email e senha)
- Design simples e limpo
- Conteúdo em Hindi

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
